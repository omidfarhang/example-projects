# Realtime Frontend Patterns

Companion project for [Real-Time Data in Frontend Applications](https://omid.dev/2024/06/08/real-time-data-in-frontend-applications/).

Local demo comparing WebSocket and Server-Sent Events side by side.

## Run

```bash
npm install
npm start
```

Open http://localhost:3000

## What it demonstrates

- WebSocket connection, send/receive, and server broadcast
- SSE stream with automatic reconnect behavior
- Simple static client without a frontend framework

GraphQL subscriptions, gRPC streams, and HTTP push are covered conceptually in the blog post and can be added later if needed.
