Angular workspace

Shared library in a monorepo

my-shared-ui is a publishable library consumed by demo-app through TypeScript path mapping — the same setup you'd use before publishing to npm.

How the workspace fits together

  1. Library project projects/my-shared-ui Standalone component, directive, and service exported from public-api.ts
  2. Build output ng build my-shared-ui ng-packagr writes consumable bundles to dist/my-shared-ui
  3. Demo application import from 'my-shared-ui' Path alias in root tsconfig.json points at the built library

Public API surface

Component MySharedUiComponent Selector: lib-my-shared-ui
Directive HighlightDirective Attribute: libHighlight
Service MySharedUiService providedIn: 'root'
