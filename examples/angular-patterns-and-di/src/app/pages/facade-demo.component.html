Design pattern

Facade over DataService

The component talks to DashboardFacade instead of DataService directly. The facade exposes a narrow API while reusing the same observable underneath. 

Component → DashboardFacade → DataService → BehaviorSubject
Value via facade.data$ {{ facade.data$ | async }}

Same data: if you updated the stream on the Observer route, that value appears here too — one singleton service, two presentation patterns.
