Web Components

Stencil component in Angular

A Stencil-built custom element works in Angular like any native HTML tag. Properties flow in, DOM events flow out — no wrapper component required.

How the pieces connect

  1. Stencil library stencil-my-button Compiles to standards-based custom elements
  2. Loader defineCustomElements() Registers <my-button> in the browser
  3. Angular host CUSTOM_ELEMENTS_SCHEMA Lets Angular templates use unknown tags

Component contract

@Prop() text Bound as [text]="buttonText"
@Event() buttonClick Listened as (buttonClick)="onButtonClick()"
