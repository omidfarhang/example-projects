Angular performance

Web Workers + OffscreenCanvas

Heavy math and canvas rendering run off the main thread so Angular can keep the UI responsive. Watch the heartbeat counter — it keeps ticking while workers do the expensive work.

Three threads in this demo

  • Main thread Angular UI Controls, layout, heartbeat counter
  • Web Worker Compute worker 1M-iteration √i loop
  • Web Worker Canvas worker OffscreenCanvas animation
