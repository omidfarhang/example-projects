# TypeScript Advanced Types

Companion project for [Deep Dive into Advanced TypeScript](https://omid.dev/2024/06/14/advanced-typeScript-types/).

Pure TypeScript examples for conditional types, mapped types, recursive types, and utility-style transformations.

## Run

```bash
npm install
npm run typecheck
```

## Files

| File | Topic |
|------|-------|
| `src/conditional-types.ts` | Conditional types, type guards |
| `src/mapped-types.ts` | ReadonlyDeep, Optional, Stringify |
| `src/recursive-types.ts` | Json, TreeNode, DeepPartial |
| `src/utility-types.ts` | PickByType, OmitByType, NonNullableFields |
| `src/*.test.ts` | Type-level checks with `@ts-expect-error` |
