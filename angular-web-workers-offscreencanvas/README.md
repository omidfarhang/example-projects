# Angular Web Workers and OffscreenCanvas

Companion project for [Optimizing Angular Applications with Web Workers and OffscreenCanvas](https://omid.dev/2024/06/23/optimizing-angular-applications-with-web-workers-and-offscreencanvas/).

## Run

```bash
npm install
npm start
```

Open http://localhost:4200

## What it demonstrates

- Offloading heavy computation to a Web Worker
- Rendering an animation on an `OffscreenCanvas` in a separate worker thread
- Start/stop controls for the canvas animation
