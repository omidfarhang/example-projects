What this version demonstrates

Tailwind moves the design decisions closer to the markup.

The brand color, rounded cards, soft shadows, responsive spacing, and status treatments are visible in the classes or the small Tailwind theme extension instead of being split across Bootstrap and custom CSS.

Replace component classes

Map Bootstrap cards, badges, buttons, grid, and progress bars to utilities.

Promote repeated values

Keep brand colors and shadows in Tailwind config when they repeat.

Keep breakpoints explicit

Use responsive prefixes where the Bootstrap grid used columns.

Delete CSS last

Remove Bootstrap and legacy overrides after parity checks pass.
